## ANSI Color Codes Reference

For terminal rendering, these examples use:
- `\033[36m` - Cyan (headers)
- `\033[32m` - Green (positive/healthy)
- `\033[33m` - Yellow (warnings/moderate)
- `\033[31m` - Red (alerts/breaking)
- `\033[90m` - Gray (borders)
- `\033[0m` - Reset
